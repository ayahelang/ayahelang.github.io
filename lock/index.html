🔒 Silverhawk Lock Guide

Panduan cepat memasang lock.css dan lock.js ke semua website HTML/CSS/JS Anda.

1. Simpan File di Server

Letakkan file berikut di folder:

/lock/lock.css /lock/lock.js /lock/how.html

Contoh URL:

  • https://silverhawk.web.id/lock/lock.css
  • https://silverhawk.web.id/lock/lock.js
  • https://silverhawk.web.id/lock/how.html

2. Tempel ke File HTML

Tambahkan CSS di dalam tag <head>.

<link rel="stylesheet" href="https://silverhawk.web.id/lock/lock.css">

Tambahkan JS tepat sebelum tag </body>.

<script src="https://silverhawk.web.id/lock/lock.js"></script>

3. Letak Baris yang Tepat

  • index.html → sebelum </head> untuk CSS
  • index.html → sebelum </body> untuk JS
  • Jika banyak halaman, tempel di semua file HTML

4. Contoh Lengkap

<head>
  ...
  <link rel="stylesheet" href="https://silverhawk.web.id/lock/lock.css">
</head>
<body>
  ...isi website...
  <script src="https://silverhawk.web.id/lock/lock.js"></script>
</body>

7. Detail Fitur Lock System

  • Blok Klik Kanan → mencegah menu context browser muncul.
  • Blok Shortcut Inspect → menahan F12, Ctrl+Shift+I, Ctrl+U, Ctrl+S.
  • Promo Redirect Random → saat trigger tertentu, membuka salah satu sosial media secara acak.
  • Floating Follow Button → tombol Follow Us selalu tampil di layar.
  • Toast Notification → pesan kecil muncul saat user mencoba aksi tertentu.
  • Anti Drag → mencegah drag gambar/link tertentu.
  • Anti Select Text → mempersulit seleksi teks.
  • Watermark Brand → label Silverhawk Network di sudut layar.
  • Zero Conflict Namespace → variabel memakai prefix unik agar aman dipasang di banyak website.
  • Central Update System → update 1 file lock.js, semua website ikut berubah.

6. Catatan

Jika website memiliki fungsi klik kanan penting, gunakan versi lock yang lebih ringan atau whitelist halaman tertentu.
