2. Cara Pasang ke Website

Tempel di dalam <head>

<link rel="stylesheet" href="https://silverhawk.web.id/lock/lock.css">

Tempel sebelum </body>

<script src="https://silverhawk.web.id/lock/lock.js"></script>

3. Setting di Tag Body

<body
data-shlock-mode="ultimate"
data-shlock-domain="eyecare.silverhawk.web.id,localhost"
data-shlock-brand="EyeCare"
data-shlock-social="https://instagram.com/a,https://youtube.com/b"
>

4. Pilihan Mode

lite

  • blok klik kanan
  • blok shortcut inspect
  • watermark

normal

  • fitur lite
  • promo sosial random
  • anti drag

aggressive

  • fitur normal
  • anti iframe
  • detect devtools

ultimate

  • semua fitur aktif
  • floating Follow Us
  • anti select
  • proteksi maksimal
